## Overview

This project is a starter kit for building agentic workflows with a clean separation between planning, execution, and verification. It includes a baseline routing layer, memory interfaces, and evaluation hooks to measure task quality.

## Highlights

- Pluggable agent roles (planner, executor, verifier)
- Memory adapters for local or external storage
- Traceable task execution and evaluation logs

## Tech Stack

- TypeScript
- Astro content collections (for docs)
- Modular task runner
